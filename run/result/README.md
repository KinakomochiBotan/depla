# 結果

実験結果です。

それぞれのフォルダに以下の4つのファイルが入っています。

例えば実験3.3であれば[3/3][3.3]に結果があります。


## cnn.pt

[CNNクラス][CNN]のオブジェクトが保存されています


## result.txt

上から順番に以下のデータが記載されています。

* 学習時間(hours:minutes:seconds)
* テストデータ1での正解率
* テストデータ2での正解率
* ランダムプレイヤーとの対戦結果(黒、win,draw,lose)
* ランダムプレイヤーとの対戦結果(白、win,draw,lose)


## loss.png

学習時の損失の変化です。


## accuracy.png


学習時の正解率の変化です。
